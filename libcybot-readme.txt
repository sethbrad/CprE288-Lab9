
libcybotUART.txt is a pre-compiled library for CyBot/Human UART communication. 
The file extension must be changed from .txt to .lib after copying into your folder.
	NOTE: This is a new library with new functions compared to the library provided for the UART part in Lab 3. See the header file. In Lab 4, use these new cybotUART header and library files.

libcybotScan.txt is a pre-compiled library for CyBot sensor scanning. 
The file extension must be changed from .txt to .lib after copying into your folder.
